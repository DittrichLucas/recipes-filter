## Requirements
- :heavy_check_mark: **Node** >= 8
- :heavy_check_mark: **Yarn** (or NPM)

----------------------
## Installation

Clone this repo, install the dependencies and start the app.

> git clone https://github.com/DittrichLucas/recipes-filter.git

> yarn

> yarn start
